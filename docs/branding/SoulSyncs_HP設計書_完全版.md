# 1. プロジェクト概要

## 1.1 目的

現在のHPを一新し、以下を達成する：
- **「何屋さんかわからない」問題を解決**
- **Soul Sync（AI業務支援）をメインに据える**
- **カズさんのストーリーで信頼を獲得**
- **問い合わせ（無料診断）につなげる**

## 1.2 ターゲット

| 項目 | 内容 |
|------|------|
| **業種** | 中小企業（特に千葉県） |
| **規模** | 従業員10〜200名程度 |
| **決裁者** | 社長・経営者（50〜65歳） |
| **課題** | 人手不足、属人化、デジタル化の遅れ、社長が忙しすぎる |

## 1.3 コンバージョン（ゴール）

**「無料診断を申し込む」ボタンをクリックしてもらう**

| 名前 | コード | 用途 |

| 項目 | 方向性 |

### コピー

**見出し：**
```
ソウルシンクスが、御社の「脳みそ」と「手足」になります。
```

**【脳みそ】Soul Sync：**
```
AIが社長の分身となり、考え、判断し、指示を出す。

・情報集約：社内の情報を一元化し、社長に届ける
・発信支援：社長の思いを、全員に届ける
・タスク管理：やるべきことを管理し、漏れをなくす
・伴走支援：一人ひとりの業務をサポート
```

**【手足】人的サービス：**
```
AIではできない「人の仕事」は、私たちが代行します。

・営業代行：リスト作成から訪問・クロージングまで
・マーケティング：SNS運用、動画制作、採用支援
・クリエイティブ：HP制作、LINE構築、デザイン
・財務支援：補助金申請、財務相談、経費最適化
```

### デザイン指示

- 2つの大きなカード（脳みそ / 手足）で対比を見せる
- 「＋」マークで「組み合わせ」を表現
- 各機能は小さなカードまたはアイコン＋テキストで表現
- Soul Syncの方を大きく・目立たせる（メイン商品なので）

### コピー

**見出し：**
```
導入の流れ
```

**ステップ：**
```

御社に最適な導入プランをご提案します。

Step 2: 構築
御社専用のSoul Syncを設計・実装します。
既存のツールとの連携も行います。

Step 3: 運用
定着するまで伴走支援します。
効果測定・改善を継続的に行います。
```

- 矢印でフローを表現
- 各ステップに数字（01, 02, 03）を大きく表示
- シンプルなアイコンを添える
